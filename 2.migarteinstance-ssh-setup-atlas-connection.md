
mkdir -p ~/.ssh
chmod 700 ~/.ssh

vi  ~/.ssh/authorized_keys

chmod 600 ~/.ssh/authorized_keys
chmod 700 ~/.ssh


Open config:

sudo nano /etc/ssh/sshd_config

Make sure these lines exist:

Port 22
PermitRootLogin no
PasswordAuthentication yes   # (only if you want password login)
PubkeyAuthentication yes


--- sudo systemctl restart ssh


Allow EC2 IP in MongoDB Atlas

Go to:
MongoDB Atlas → Network Access → IP Access List 15.207.23.11/32

Get MongoDB connection string
In Atlas:

Go to Clusters → Connect → Drivers
copy URI like:
mongodb+srv://<username>:<password>@cluster0.xxxxx.mongodb.net/dbname

.env:

MONGO_URI=mongodb+srv://user:pass@cluster0.xxxx.mongodb.net/dbname

test - node test-db.js
